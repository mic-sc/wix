# International Space Station - upcoming fly 

The International Space Station (ISS) is an orbital outpost circling high 
above our heads.

This project retrieves data from the ISS API and stores the information in 
a database, runs a procedure that calculates the average daily runs per city 
and generates a csv file with the city details and stats.

######IMPORTANT: None
